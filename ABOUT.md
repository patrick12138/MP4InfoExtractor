# About MP4InfoExtractor

## English

MP4InfoExtractor is a lightweight WPF desktop utility for Windows that scans video files and extracts structured song metadata from filename rules. It is designed for batch media organization and CSV export workflows.

Core goals:
- Fast folder-level batch scanning
- Consistent filename parsing into structured fields
- Simple CSV export for downstream processing

## 中文

MP4InfoExtractor 是一个轻量级 Windows WPF 桌面工具，用于扫描视频文件，并从文件名规则中提取结构化歌曲信息。它主要用于批量媒体整理与 CSV 导出流程。

核心目标：
- 快速批量扫描目录
- 将文件名稳定解析为结构化字段
- 提供简单可用的 CSV 导出能力
